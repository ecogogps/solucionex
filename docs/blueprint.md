# **App Name**: Hola Mundo Display

## Core Features:

- Display Hello World: Render the text 'Hola Mundo' prominently on the screen.

## Style Guidelines:

- Primary color: A calm, reassuring blue (#0D0D54), chosen to evoke clarity and a clean start.
- Background color: A very light, almost imperceptible grey-blue (#2E2E2E), offering a neutral canvas for readability within a light theme.
- Accent color: A vibrant, clear cyan (#FFFFFF), providing a fresh and energetic contrast to draw attention where needed, derived analogously from the primary hue.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, objective, and neutral appearance, ensuring maximum legibility for simple text display.
- Centered display: The 'Hola Mundo' text will be centrally aligned for clear focus and a minimalist aesthetic.
